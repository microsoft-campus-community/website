Introduction to Kubernetes

Language: English

The goal of the hands-on course is to take a small local deployment and run it on Kubernetes, giving insight into different deployment tools and options. To start off the workshop we will be using Kompose to create a simple deployment, followed by Kustomize and finally the creation of a small Helm chart. You will need a basic understanding of Kubernetes to follow this talk.
