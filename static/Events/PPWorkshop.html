Low-Code / No-Code Development for All!

Lee Zuckett, Program Manager for the Power Platform, will teach us how to build no-code / low-code solutions.

The next wave in software development has nothing to do with coding.

According to Microsoft, 500 million new apps will be built in the next 5 years. This is more than the number of apps built in the last 40 years.
This number of apps will not be built by traditional developers. These apps will be built by Nurses, Architects, Biologists, Visionaries, ..., [insert your profession]. Join us to learn more about no-code / low-code development.

Agenda:

  • Introduction Low-Code / No-code
  • Introduction Power Platform / Power Apps
  • Walk-through
  • Q & A
